# GEISA schemas

This repository contains the [protobuf](https://protobuf.dev/) or
[JSON Schema](https://json-schema.org/) schemas used by the
[Grid Edge Interoperability and Security Alliance Specification](https://github.com/geisa/specification).

These schemas define the message structures used by the GEISA API and message bus defined in the specification.

For full documentation and context, refer to the [GEISA Specification][geisa-spec] for additional details.

[geisa-spec]: https://github.com/geisa/specification
